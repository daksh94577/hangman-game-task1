# Hangman Game - Task 1

This is a simple text-based Hangman game built using Python.

In this game, the player has to guess a hidden word one letter at a time.  
The player gets a maximum of 6 incorrect guesses before the game ends.

## Features
- Small list of 5 predefined words
- Random word selection
- 6 maximum incorrect guesses
- Simple console-based input/output
- Letter-by-letter word guessing

## Concepts Used
- random module
- while loop
- if-else statements
- strings
- lists
